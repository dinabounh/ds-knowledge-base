## 📑 Repository Format
This repository acts as my "Central Brain 🧠" focusing on theory and logic. Project implementations are linked to my portfolio. 

### Data Tools 🛠️
- **SQL**: MySQL environment showcasing a collection of optimized queries, window functions, and schema design patterns for data retrival.
- **Tableau and PowerBI**: Showcases dashboard format examples. 

### Linear Algebra 🔢
- **LaTex `.txt`files**: Used for rigorous mathematical foundations and notations.

### Machine Learning ⚙️
- **Jupyter Notebook `.ipynb` files**: Used for interactive ML implementations using Scikit-Learn, Tensorflow, and PyTorch.

### Statistics 📊
- **Plain Text `.md`file Notes**: Used for probability theories/distributions and statistcal test including t-test, z-test, ANOVA, and Chi-squared.
- **R Markdown`.rmd` files**: Used for statistical analysis using Tidyverse and Stats.
- **Jupyter Notebook `.ipynb` files**: Used for statistical computations using Numpy, Pandas, Scipy.

### Simulation 🔍
- **Jupyter Notebook `.ipynb` files**: Used to showcase linear and integer programming for optimization problems through PuLP.
- **R Markdown `.rmd` files**: These file notes include examples of Monte-Carlo Simulation and Discrete Event Simulation examples through simmer library.
